# BWhaleMonitor-PanelBackend

BWhaleMonitor 蓝鲸服务器探针主控面板后端仓库


## 本项目由以下工具/项目提供支持
- JetBrains IDE (PhpStorm)
- 通义灵码
- WorkerMan
  - workerman/webman-framework
  - workerman/validation
- Laravel
  - laravel/serializable-closure
- illuminate
  - illuminate/events 
  - illuminate/database
  - illuminate/pagination
- symfony
  - symfony/var-dumper
- monolog
  - monolog/monolog
